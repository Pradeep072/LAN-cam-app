# Project Roadmap & Issues

This document tracks planned enhancements and known areas for improvement. Contributors are encouraged to pick up any of these tasks!

## High Priority: Video Quality & Performance
- [x] **Improved Video Quality**: Optimized VLC caching and hardware acceleration settings to reduce latency.
- [x] **Multi-Camera Optimization**: Improved resource management with centralized VlcManager.
- [x] **Adaptive Bitrate Support**: Added HLS/DASH support for variable network conditions.

## UI/UX Enhancements
- [ ] **TV Navigation**: Refine D-pad navigation for better accessibility on Android TV.
- [ ] **Camera Auto-Discovery**: Enhance the scanning logic to reliably detect more camera brands.
- [ ] **Custom Grid Layouts**: Allow users to customize the size and position of camera feeds in the multi-view.

## General Improvements
- [ ] **Unit & UI Tests**: Increase test coverage for camera discovery and database operations.
- [ ] **Localization**: Add support for more languages.

## Help Wanted
If you are new to the project, look for these tasks:
- **UI Tweaks**: Improving the look and feel of the camera list.
- **Documentation**: Adding more detailed comments to the core streaming logic in `MultiViewPlayerActivity.kt`.
